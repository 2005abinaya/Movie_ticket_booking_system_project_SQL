🎬 Movie Ticket Booking System (MySQL Project)
📌 Project Overview

The Movie Ticket Booking System is a relational database project developed using MySQL.
This system simulates a real-world online movie ticket booking platform where users can:

Browse movies

View show timings

Book seats

Make payments

Apply offers

Give reviews

The project demonstrates strong knowledge in:

Database Design

Table Relationships

Joins

Aggregation Functions

Subqueries

Scenario-Based SQL Queries

🏗️ Database Design

The system consists of 15 relational tables:

Users

Admin

Movies

Theatres

Screens

Shows

Seats

Bookings

Booking_Seats

Payments

Reviews

Offers

Applied_Offers

Cities

Theatre_City_Map

All tables are connected using Primary Keys and Foreign Keys to maintain data integrity.

🗄️ Database Structure
🔹 Core Functional Flow

User → Books Show → Selects Seats → Makes Payment → Applies Offer → Gives Review

This flow is implemented using proper relational mapping between tables.
